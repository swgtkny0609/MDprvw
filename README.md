# マークダウン・ノート（PWA版）

オフラインで動く、上品なマークダウン・ビューア／エディタです。
ネット上に置くと、ブラウザから「アプリ」としてインストールできます。

## 入っているファイル
- `index.html` … アプリ本体
- `manifest.webmanifest` … アプリ情報（名前・アイコン・色）
- `sw.js` … オフライン動作の仕組み
- `icon-192.png` / `icon-512.png` / `icon-maskable-512.png` … アイコン

※ PWAは安全のため **https のURL** でしか正式インストールできません。
　ファイルを直接ダブルクリック（file://）ではインストールできないので、
　まず下のどちらかの方法でネット上に置いてください。

---

## 方法A：Netlify Drop（いちばん手軽・ドラッグするだけ）
1. ブラウザで https://app.netlify.com/drop を開く
2. このフォルダ（中のファイル一式）を、ページ上にドラッグ＆ドロップ
3. しばらくすると `https://〇〇〇.netlify.app` というURLが発行される
4. そのURLを Edge か Chrome で開く

## 方法B：GitHub Pages（無料・URLを固定したい人向け）
1. GitHub でリポジトリを新規作成
2. このフォルダのファイルをすべてアップロード
3. Settings → Pages → Branch を `main` / フォルダ `root` にして保存
4. 数分後に発行される `https://ユーザー名.github.io/リポジトリ名/` を開く

---

## インストール（Windows）
1. 発行された https のURLを Edge または Chrome で開く
2. アドレスバーの右に出る「インストール」アイコン（画面＋下矢印のマーク）をクリック
   - 見当たらないときは「⋮」メニュー →「アプリ」→「このサイトをアプリとしてインストール」
3. 「インストール」を押す

これで、スタートメニュー・タスクバーに追加され、タブのない専用ウィンドウで開きます。
一度開いたあとは、ネットがなくてもそのまま使えます（自動保存つき）。

## アンインストール
Windowsの「設定 → アプリ」から、または アプリのウィンドウ右上「⋮」→「アンインストール」。

---

## エクスプローラーの .md を、常にこのアプリで開く

このPWAは `.md` / `.markdown` / `.txt` を担当できるようになっています。
手順は次のとおりです（Edge / Chrome で一度アプリをインストール済みであること）。

1. 公開URLを開き、アプリを **インストール（または再インストール）** する
   ※ 以前の版を入れている場合は、一度アンインストールしてから入れ直すと確実です
   （アプリのウィンドウ右上「⋮」→「アンインストール」）
2. インストール時に「このアプリでファイルを開くのを許可しますか？」と聞かれたら「許可」
3. エクスプローラーで `.md` ファイルを右クリック →「プログラムから開く」→「別のプログラムを選択」
4. 一覧から「マークダウン・ノート」を選び、
   「常にこのアプリを使って .md ファイルを開く」にチェック →「OK」

以降は、`.md` をダブルクリックすると、このアプリで中身が開きます。

### うまく出てこないとき
- Windowsの「設定 → アプリ → 既定のアプリ」で `.md` を検索し、
  ファイルの種類ごとに「マークダウン・ノート」を選ぶ方法でも設定できます。
- 一覧にアプリが出ない場合は、アプリの再インストール後にもう一度お試しください。

## ファイルのアイコンが白い（無地）になるとき

エクスプローラー上で `.md` ファイルのアイコンが白くなるのは、
ブラウザからインストールしたPWAでよく起きる、現状の仕様上の制限です。
（ファイルを開く関連付けはできていても、ファイル自身のアイコンは
Windows側で確実には差し替わらないことがあります。）

このパッケージでは、Windowsが使いやすいサイズのアイコン
（16 / 32 / 48 / 256）をマニフェストに用意してあります。
対応しているWindows・Edge/Chromeのバージョンなら、これでアプリの
アイコンが表示されます。反映には次を試してください。

1. アプリを一度アンインストールし、公開URLから入れ直す（登録し直すため）
2. パソコンを再起動、またはエクスプローラーを再起動する
   （アイコンの表示キャッシュを更新するため）

それでも白いままの場合は、Windows/ブラウザ側がまだ未対応のためで、
アプリの動作自体には影響ありません。

### どうしても確実にアイコンを付けたい場合（上級者向け）
同梱の `app.ico` を使って、Windowsのレジストリでアイコンを指定する方法があります。
少し踏み込んだ操作になるため、必要であればお手伝いします。
